# Global Keys

Keys that are supported by every section.

## env

Pass envs to command generated by the section.


### Example
```ini
[printenv]
cmd=printenv FOO
env=
    FOO=val
```
## via
Pass the command generated by this section into a different section.

### Example

```ini
[ssh]
cmd=ssh root@server {}

[serverstats]
cmd=htop
via=ssh
```
